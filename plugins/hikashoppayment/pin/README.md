Hikashop payment plugin for pin payment plugin

Install in /plugins/hikashoppayment/pin/

Developed for/on
----------------
Joomla! 3.8+, 4.x, 5.x and 6.x (PHP 8 compatible)
HikaShop (any edition)

Uses
----
No external libraries. Charges are sent directly to the Pin Payments
REST API: https://pinpayments.com/developers/api-reference
